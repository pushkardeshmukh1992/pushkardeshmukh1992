### 🚀 My Portfolio — [pushkardeshmukh.com](https://pushkardeshmukh.com)

I built my portfolio from scratch as a **full-stack web app** — not just a static page. It's a living platform with a blog engine, auth, real-time engagement, and automated publishing.

**What's under the hood:**

- **Blog platform** with views, likes, comments (Google Auth), and related posts
- **Admin dashboard** with a rich text editor (Tiptap), image uploads, comment moderation, and analytics
- **Auto-publishing pipeline** — publishing a post automatically shares to LinkedIn, pings Google Indexing API, IndexNow (Bing/Yandex), and generates an RSS feed
- **Discord webhooks** for real-time engagement alerts and centralized error reporting
- **Dark/Light theme**, responsive design, SEO-optimized with JSON-LD schemas

> Built with **Next.js 16** · **React 19** · **Supabase** · **Tailwind CSS** · **Framer Motion** · Deployed on **Vercel**
